### `II. Current Weather`

**Function:** 
Display current weather conditions (temperature, humidity, wind, weather icon, max/min temps) in real-time on the screen.

### `III. Forecast`

**Function:** 
Display a customizable daily weather forecast (0 ~ 14 days) with weather icon, max/min temperatures. Show max 7 items on screen; enable horizontal scroll if forecast exceeds 7 days.

[More…](https://www.notion.so/More-35e6824f615780a7a129d674713ffc20?pvs=21)

---

### `IV. Graph`

**Function:** 
Display weather data in a time-series chart. The user can select a data type from Extra Data. If a specific day is also selected, display hourly data; otherwise, display daily data across all forecast days. 

[More…](https://www.notion.so/More-35f6824f61578011aaddc71e24740793?pvs=21)

---

**`V. Weather Metrics`**

**Function:**
Display selectable data type cards (Max Temp, Min Temp, Humidity, Precipitation Chance, Precipitation Sum, Wind Speed). The user can click to select which data type to display in the Graph. In hourly mode, also allow day selection.
